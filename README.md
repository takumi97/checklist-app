# Checklist App

Apple-style minimalist checklist app built with:
- Next.js
- Tailwind CSS
- shadcn/ui
- Framer Motion
- Lucide React

## Getting Started

### 1. Install dependencies
```bash
npm install
```

### 2. Run development server
```bash
npm run dev
```

### 3. Deploy to Vercel
Connect this project to your Vercel account and deploy.

# Checklist App (Apple-style)

This is a minimalist, Apple-inspired checklist app built with:

- Next.js 14
- Tailwind CSS
- shadcn/ui
- Lucide React
- Framer Motion

## Local Setup

```bash
npm install
npm run dev
```
